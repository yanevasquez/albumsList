#### 1. Create a directory and clone the repository
```sh
$ mkdir app
$ cd app
$ git clone https://github.com/yanevasquez/albumsList
```
#### 2. Building the **albumList-app** container 
```sh
$ docker-compose up -d
```
 App running at `http://localhost:8081`
